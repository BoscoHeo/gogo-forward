💾 문제 세트 저장 & 공유

🔑 1단계: 선생님 등록 / 로그인

  • 첫 방문 시 새로 등록 → 이름과 학교 입력 → 교사 코드 발급
  • 이후 방문 시 기존 코드 입력 → 발급받은 6자리 코드 입력
  • ⚠️ 교사 코드를 잃어버리면 복구가 불가합니다. 꼭 기억해두세요!

📝 2단계: 문제 만들기

  • 게임 만들기 탭 → 게임 제목, 제한 시간 설정
  • 스테이지 추가 후 문제 입력 (객관식 / 주관식)
  • 한글에서 붙여넣기 버튼으로 표 형식 문제 일괄 가져오기 가능
  • 팀 대결 모드 ON → 팀 수(2~6) 및 팀 이름 설정 가능

🚀 3단계: 게임 진행

  • 게임 코드를 학생에게 공유 → 학생들이 입장
  • 대시보드에서 학생 진행 상황 실시간 확인
  • 팀 대결 시 학생 카드의 드롭다운으로 팀 배정 가능
  • 게임 시작 버튼을 눌러야 학생 화면이 시작됩니다

🤖 AI 채점 (선택사항)

  • Gemini API 키 입력 시 주관식 유사 답안 자동 인정
  • API 키는 이 PC에만 저장 (서버 전송 없음)
  • 선생님 대시보드가 열려 있는 동안 AI 채점 작동
  • 키 발급: aistudio.google.com/apikey

📚 문제 라이브러리

  • 만든 문제 세트를 다른 선생님과 공유 가능
  • 공개 라이브러리에서 다른 선생님 문제 가져오기 가능
  • 삭제는 만든 선생님만 가능, 수정은 복사본으로 저장

🎮 1단계: 게임 참여

  • 메인 화면에서 학생 → 게임 참여하기 클릭
  • 선생님이 알려준 게임 코드 입력
  • 내 이름 입력 → 팀 선택(팀 대결 시) → 입장하기
  • 선생님이 게임을 시작할 때까지 대기

🏁 2단계: 문제 풀기

  • 스테이지 순서대로 문제 풀기 (1 → 2 → 3 → ...)
  • 객관식: 4개 보기 중 정답 선택 후 정답 제출!
  • 주관식: 답 입력 후 정답 제출! (오타 1~2개 허용)
  • 힌트(예: ○○→○○)가 표시되면 정답 형식 참고

⚡ 3단계: 정답 / 오답

  • ✅ 정답: 다음 스테이지로 자동 이동
  • ❌ 오답: 1스테이지 뒤로 이동 (1단계에서 틀리면 1단계 유지)
  • 제한 시간 내에 최대한 높은 스테이지까지 도달하는 것이 목표!

📝 게임 종료 후

  • 게임 종료 후 내가 푼 문제 전체 복습 화면 표시
  • ✅ 정답 문제 / ❌ 오답 문제 + 정답 확인 가능
